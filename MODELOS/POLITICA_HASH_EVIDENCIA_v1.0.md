POLÍTICA HASH EVIDENCIA CANON v1.0

Estado: VIGENTE
Versión: 1.0.0

Principio

Toda evidencia almacenada en /EVIDENCIA debe tener hash SHA-256 registrado.

Reglas

No se permite evidencia sin hash.

El hash debe registrarse en:

Archivo dentro de /HASH

O dentro de la bitácora correspondiente.

Formato obligatorio:

ARCHIVO: nombre.ext
SHA256: valor_hash
FECHA: YYYY-MM-DD

Si no existe hash verificable → Estado mínimo PARCIAL.
